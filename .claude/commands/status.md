Use the project-manager agent to generate a project status report.

Check:
1. **Progress** - Review claude-progress.txt and feature-status.json
2. **Active Work** - Check AGENTS.md for claimed files
3. **Build Health** - Quick build/type/lint check
4. **Issues** - Any blockers or concerns

Report format:
- Overall progress (X% complete)
- Active tasks
- Blocked tasks
- Next priorities
- Any issues needing attention
